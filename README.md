# GRaWACpy

GRaWACpy is a Python software to process RPG G-band Radar for Water vapor and Arctic Clouds (GRaWAC) data. More information on the instrument can be found in [Schnitt et al., 2025, AMT](https://egusphere.copernicus.org/preprints/2025/egusphere-2025-5563/).

The software features reading different output formats, matching W- and G-band radar measurements in time and range, correcting radar measurements for gas attenuation, and retrieving water vapor profiles from GRaWAC measurements. Additional quicklooks are produced to give an overview of measurements.
